Web Development Basics

HTML, CSS, JavaScript의 세계에 오신 것을 환영합니다

1. 웹 개발의 3가지 핵심 언어

웹 페이지를 만드는 과정은 '집 짓기'에 비유하면 매우 쉽습니다.

  • 🏗️ HTML (뼈대): 기둥을 세우고 골조를 잡는 과정입니다. 제목, 본문, 버튼 등이 어디에 위치할지 정의합니다.
  • 🎨 CSS (인테리어): 벽지를 바르고 가구를 배치하는 과정입니다. 색상, 글꼴, 레이아웃 등 시각적 디자인을 입힙니다.
  • ⚡ JavaScript (전기와 근육): 전기를 연결해 불이 켜지게 하듯, 정적인 페이지를 살아 움직이게(동적으로) 만듭니다.

2. HTML 기본 문서 구조

모든 HTML 파일은 다음과 같은 표준 형식을 갖추어야 브라우저가 올바르게 해석합니다.

<!DOCTYPE html>
<html lang="ko">
  <head>
    <title>페이지 제목</title>
  </head>
  <body>
    <h1>실제 화면에 나오는 내용</h1>
  </body>
</html>

3. 자주 사용하는 웹페이지 요소 (Tag)

HTML 태그를 사용하여 정보를 구조화하는 방법입니다.

태그 설명 사용 예시
<h1>~<h6> 중요도에 따른 제목 표시 가장 큰 제목은 h1
<p> 본문 문단 (Paragraph) 설명 글을 적을 때 사용
<a> 링크 (Anchor) 다른 페이지로 이동
<img> 이미지 삽입 사진이나 아이콘 표시
<ul>, <li> 목록 (List) 리스트 형태의 항목 정리
<button> 클릭 가능한 버튼 이벤트 실행 시 사용

파일을 연결할 때 사용하는 경로 지정 방법입니다.

  • ./04/video.mp4 : 현재 폴더 안의 04 폴더 내 영상 파일
  • ../index.html : 현재 위치에서 한 단계 위로 나가서 찾는 홈 파일
